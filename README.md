## Minimal Path to Awesome

- Clone this repository
- Ensure that you are at the solution folder
- in the command-line run:
  - **npm install**
  - **npm run start**


## Features

This repository demonstrates how to use Aurelia 2 with the new spfx heft build pipeline. Specifically that components, dependency injection and styles works as expected.
The example shows themed module.scss, regular scss, simple css and component css.
